# Competitor Analysis — Checkjouwdomein.nl

## Directe Concurrenten (domeinchecker / vergelijkingssites)

### Domainchecker.nl
- **URL**: domainchecker.nl
- **Focus**: Basis domeinchecker
- **Zwakte**: Geen prijsvergelijking tussen registrars
- **Onze edge**: Wij tonen welke registraar goedkoopst is

### Hosting.nl domeinchecker
- **URL**: hosting.nl
- **Focus**: Hosting + domein bundels
- **Zwakte**: Verkopen eigen hosting, niet onafhankelijk
- **Onze edge**: Volledig onafhankelijk, meerdere registrars

### Domeinservice.nl
- **URL**: domeinservice.nl
- **Focus**: WHOIS en domeinregistratie
- **Zwakte**: Verouderde UI, geen moderne vergelijking

## Indirecte Concurrenten (registrars zelf)

### TransIP
- Sterkste NL-registraar qua naamsbekendheid
- Eigen blog met domein-tips
- Wij zijn onafhankelijk, zij niet

### Mijndomein.nl
- Goedkoopste introductieprijs
- Eigen vergelijkingstools
- Wij tonen verborgen verlengingsprijzen

## Content Gap Analyse

### Onderwerpen waar concurrenten goed op scoren maar wij niet
1. "TransIP ervaringen" → Kans voor review artikel
2. "Beste domeinregistraar nederland" → Pillar content
3. "Domein verhuizen stap voor stap" → How-to artikel
4. "WHOIS opzoeken" → Tool + uitleg

### Onderwerpen waar wij uniek kunnen zijn
1. Real-time prijsvergelijking (onze tool)
2. Transparantie over verlengingsprijzen
3. Nederlandse onafhankelijke reviews
